# Neovim C++ Competitive Programming Workspace

This repository contains the complete configuration for a high-performance C++ competitive programming environment in Neovim.

## 📁 Repository Structure
- **[README.md](README.md)**: Plugin integration and overall setup guide.
- **[snippets/cpp.lua](snippets/cpp.lua)**: The core snippet definitions (Templates, DSU, Math, STL).
- **[SHORTCUTS.md](SHORTCUTS.md)**: Guide on using snippets and type-mirroring features.
- **[MAPPINGS.md](MAPPINGS.md)**: Compilation and Execution shortcuts for rapid testing.

## 🛠️ Quick Setup
1. Copy `snippets/cpp.lua` to `~/.config/nvim/lua/custom/snippets/`.
2. Update your `LuaSnip` and `nvim-cmp` config as shown in the [README](README.md).
3. Add the compile/run shortcuts from [MAPPINGS.md](MAPPINGS.md) to your `mappings.lua`.
